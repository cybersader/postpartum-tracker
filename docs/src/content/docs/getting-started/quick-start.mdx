This guide walks you through logging your first entries. The plugin is designed for one-handed, one-tap operation -- perfect for holding a baby.

## Insert a tracker

<Steps>

1. Open a note (or create a new one, e.g., "Baby Tracker").

2. Run the command **Insert postpartum tracker** from the command palette, or tap the ribbon icon.

3. A code block appears with the tracker widget rendered inside it.

</Steps>

The widget shows several sections: a baby info bar at the top, quick action buttons, and collapsible sections for each tracker module.

## Log a feeding

Tap the **Breast L**, **Breast R**, or **Bottle** quick action button. A feeding entry is created with a live timer that counts up. Tap the stop button when the feeding ends.

The daily summary at the top updates immediately with the feeding count and time since last feeding.

## Log a diaper change

Tap **Wet**, **Dirty**, or **Both** in the quick actions. If you enabled the color picker in settings, you can select the stool color for dirty diapers.

The daily summary shows today's diaper counts. A health alert appears if the wet diaper count drops below the configured threshold (default: 6).

## Log a medication dose

Tap a medication button (e.g., **Tylenol**). The dose is recorded with a timestamp and a timer bar shows how long until the next safe dose.

If alternating medication alerts are enabled, the plugin tracks the Tylenol/Ibuprofen pattern and suggests which to take next.

:::tip[Hold for more options]
Long-press (~500ms) any quick-action button to open a form for optional fields like notes, feeding position, or stool color. Quick tap logs with defaults immediately.
:::

## Quick entry (type to log)

Prefer typing? Enable **Show quick entry** in Settings > Display. A text input appears at the top of the widget where you can type phrases like `fed left 20 min` or `wet diaper`. A live preview shows what will be logged -- press Enter to confirm.

See the [Quick entry guide](/postpartum-tracker/guides/quick-entry/) for the full keyword reference.

## Add a note or comment

Enable the **Notes & comments** tracker in Settings > Trackers. Tap "Add note" to log a timestamped free-text entry. Notes can be categorized as general, concern, milestone, or reminder.

You can also type any unrecognized text in quick entry -- it routes to comments automatically.

## Set up baby info

Tap the pencil icon in the baby info bar at the top of the widget, or open Settings > Postpartum Tracker:

- **Baby name** -- displayed in the info bar
- **Birth date** -- used to calculate day of life, weeks, and logic pack milestones
- **Birth weight** -- for reference
